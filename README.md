# Shopify-Clone

> This is a clone of [Shopify's](https://www.shopify.com/in) home page made by me.

### Tech used:
- HTML
- CSS
- Tailwindcss
- Node.js

### Mobile responsive
It is fully responsive on all mobile devices!

### Time taken
4 hours
